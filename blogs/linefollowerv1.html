Line Follower project
A project that me and my friends made for Robotics Club competitions in IIIT
Jan 2024
Introduction

The project was made in the first year of college. We made it using a nano and an ir array. It is a very simple project, using a PID controller implemented using arduino nano to control the speed of the motors and hence follow the line using the ir sensor data.
